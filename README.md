# events-manager
Мини-плагин для управления событиями без сторонних решений.

-создать перечень событий в админке, вывести на странице shortcode: [events_list]

- плагин выводить последние 3 события(post type), и загружает по 1 еще по нажатию кнопку "Показать Больше", если событий больше нету выводит надпись "Больше событий нет.", также не выводить события которые прошли.

Demo: https://optimizewp.biz/events/
